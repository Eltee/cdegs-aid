Projet:     CDEGS-Aid

Description: CDEGS-Aid est un logiciel d'aide pour la génération de fichiers de simulation compatibles avec SESCad et CDEGS-HiFreq. 
Il gère les fichiers pour des simulations de champ électrics et autres. 
Il gère aussi l'affichage et l'analyse des résultats de simulations CDEGS-HiFreq.
